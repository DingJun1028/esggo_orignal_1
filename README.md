# ESGGO Original Archive Helper

This repository currently stores the uploaded archive `ESGGO_Orignal-v25.25.249 (1).zip`.

## Unzip the archive

Use the helper script:

```bash
python scripts/unzip_esggo.py
```

Optional arguments:

- `--zip`: path to a zip file (default: `ESGGO_Orignal-v25.25.249 (1).zip`)
- `--out`: output folder (default: `extracted`)

Example:

```bash
python scripts/unzip_esggo.py --out ./workspace
```
